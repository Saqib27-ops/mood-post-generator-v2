# ✨ Mood Post Generator

An AI-powered web application that generates unique and creative social media posts based on your current mood.

## 🌐 Live Demo
👉 [mood-post-generator-v2.onrender.com](https://mood-post-generator-v2.onrender.com)

## 🚀 Features
- 🤖 AI-powered post generation using LLaMA 3.3 70B model
- 📸 Supports Instagram, LinkedIn, Twitter and Facebook
- 📋 One-click copy to clipboard
- 🕐 History of last 5 generated posts
- ⚡ Real-time character counter
- 🎨 Beautiful responsive design

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **AI Model:** LLaMA 3.3 70B via Groq API
- **Frontend:** HTML, CSS, JavaScript
- **Deployment:** Render
